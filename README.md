## 技术栈
> 本工程主要基于react16 + Mobx + Antd + webpack4 + react-router4

## 启动

```
<!--安装前请先确保已安装node和npm-->
npm start           启动项目
npm run build       发布生产版本
npm run analyz      打包分析
npm run dll         第一次build前需先运行此命令，依赖更新也要
npm run buildClean  清空/dist
npm run dllClean    清空/dll
npm run pack        index.html中写入dll.js
